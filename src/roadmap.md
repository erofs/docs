# 🗺 Roadmap

## Linux Kernel

 - Sub-page block size support of compressed files.

 - [Support fscache daemon failover mechanism to recover from crashing](https://lore.kernel.org/r/20231120041422.75170-1-zhujia.zj@bytedance.com).

## Userspace tools (erofs-utils)

 - Multi-threaded (de)compression;

 - Intel QAT/IAA accelerator support;

 - Preliminary Zstandard compression support;

 - Rebuild improvements (including incremental updates).
